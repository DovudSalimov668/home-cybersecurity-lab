# Home Cybersecurity Lab

## Overview

This repository documents my VMware-based cybersecurity laboratory built for learning:

- Linux administration
- Windows administration
- Network troubleshooting
- SSH
- Nmap
- Wireshark
- Service enumeration
- Security assessment fundamentals

The environment is fully isolated and intended only for authorized educational use.

---

## Lab Goals

1. Build a repeatable cybersecurity lab.
2. Learn networking fundamentals.
3. Practice service enumeration.
4. Learn packet analysis.
5. Document findings professionally.
6. Build a portfolio demonstrating practical skills.
